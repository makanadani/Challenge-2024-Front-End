Porto AutoAssist

FUNCIONALIDADES

Cadastro e Login

O sistema permite que novos usuários se cadastrem fornecendo informações básicas como nome de usuário, senha, nome completo, CEP, endereço, número do endereço, complemento, cidade, estado, telefone e e-mail. Usuários já cadastrados podem realizar login utilizando seu nome de usuário e senha.

Autodiagnóstico de Alterações no Automóvel

Após o login, os usuários têm acesso a uma ferramenta de autodiagnóstico que permite identificar possíveis problemas com base em sintomas comuns observados no veículo. O sistema apresenta uma lista de opções como perda de potência do motor, luzes do painel de instrumentos acesas, sons estranhos durante a condução, odor estranho dentro do veículo, lâmpada de farol apagada e veículo que não liga. Cada opção fornece uma descrição detalhada do problema e uma estimativa de orçamento para reparos.
